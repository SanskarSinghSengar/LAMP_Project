# LAMP Project – Late Antiquity Modelling

## Overview

This project implements computational modelling techniques for reconstructing embodied spatial experience at the Necropolis of El Bagawat.

### Task 1
Path tracing and movement modelling using terrain and structural data.

### Task 2
3D viewshed modelling integrating terrain, building mass, and observer height.

## Features

- 3D Digital Surface Model (DSM) construction
- Observer-based line-of-sight modelling
- Visibility probability surfaces
- Visibility entropy analysis
- 2D vs 3D comparative evaluation
